# 楞嚴神奇大法會 · 工作 SOP dashboard

寶嚴國際佛學研修院／寶嚴禪寺　·　**私有版（職稱代號）**

依「寶嚴流程表」（執事人員組織總表＋法會排程規劃表）真實內容製作的全組別工作 SOP 互動看板。

## 內容
- **總覽·組織**：總指揮中心＋15 個工作部門（代號制）、本場真實儀軌與項目
- **主時間軸 Workflow**：前置3天 → 法會3天 → 第4天三皈五戒暨菩薩戒 → 第5天圓滿撤場，每個里程碑標主責組＋協作組
- **跨組串連矩陣**：誰交給誰、交付什麼、何時觸發、走哪個管道、期限（25 條）
- **知會協定**：法會當天即時通報（12 條）＋突發三級升級（8 種）
- **壇口協作**：內壇儀軌、獻供、須彌山12座、普施、大蒙山、金剛藏王神鼓、三皈五戒暨菩薩戒正授 等關鍵時刻的多組串接（8 場景）
- **各組 SOP**：15 組五階段任務、進/出交接、風險對策、檢核清單、KPI
- **通訊頻道**：無線電頻道／LINE群組／共用雲端配置（10 個）

## 隱私
- 為保護個資，本看板**人員一律以職稱／組別代號呈現**（如 宣=宣傳組），不含個人姓名。
- 實際人員分工與名單以**總指揮中心最新公告／私版**為準。

## 使用
- 本機：雙擊 `開啟楞嚴大法會工作SOP.command`，或直接以瀏覽器開啟 `楞嚴神奇大法會工作SOP.html`。
- 手機／平板：以大字卡片式呈現，可直接瀏覽。
- 資料在 `data.js`（`window.SOP_DATA`）；版面在 `楞嚴神奇大法會工作SOP.html`。人員或流程更新時，改 `data.js` 即可，版面不用動。
